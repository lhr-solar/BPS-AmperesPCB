# BPS Amperes Board
The BPS Amperes board is a board that assists in measuring the current across a shunt resistor between the battery terminals. It outputs this data to SPI using an ADC.

#Large Components
J1 - Molex 53307-2071
J2 - 2.54mm 1x5 Pin Socket
