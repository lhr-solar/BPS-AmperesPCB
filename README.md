# BPS Amperes Board
The BPS Amperes board is a board that assists in measuring the current across a shunt resistor between the battery terminals. It outputs a filtered output to the BPS Peripheral SOM through Ain. (Pin 20)

#Dimensions
###Size = 40mm x 50mm
Sandwiched in between the BPS Peripheral SOM (Above) and Shunt Resistor (Below)

#Large Components
+ J1 - Molex 53307-2071 (SMD on front)
+ J2 - 2.54mm 1x5 Pin Socket (THT on back)
+ J3 - 2.54mm 1x5 Pin Socket (THT on back)
+ J4 - 2.54mm 1x5 Pin Header (THT on back)
+ J5 - 2.54mm 1x2 Pin Header (THT on back)
+ TP1 - 2.54mm 1x1 Pin Header (THT on back)
